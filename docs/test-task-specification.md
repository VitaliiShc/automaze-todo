# Task: Junior Full-Stack Developer

We would like you to build a TODO web application with the following functionality:

- Display a list of all tasks
- Add a new task and remove a task
- Search for tasks
- Mark a task as done
- Filter tasks by status (all / done / undone)
- Assign priority to tasks (1-10)
- Sort tasks by priority in ascending or descending order

## Tech stack

- Backend: Python FastAPI (preferred) or Node.js as a simpler alternative
- Frontend: React with Next.js and Tailwind CSS

## Expectations

- The application should have a clean, user-friendly design
- Deploy the application to a platform of your choice (for example, Vercel, Render, Railway)

## Deliverables

1. The URL of the deployed application
2. The URL of the GitHub repository

## PS: How to stand out

- This is your chance to show your best and apply all the skills you have. For example:
- Connect your app to a database (SQLite, PostgreSQL or another) instead of keeping everything in memory
- Use a component library (for example, Radix UI, shadcn/ui, Mantine or Chakra UI) to make the UI polished and modern
- Organize your code following best practices with a clear folder structure and reusable React components
- Add bonus features such as due dates, categories, or drag-and-drop for tasks

💡 The goal is not just to make the app work, but to demonstrate how you approach a real project, from clean code to thoughtful design.
